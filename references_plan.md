# References And Citation Plan

The current related-work draft uses source labels and placeholder citation markers. Before final submission, each source should be converted into a BibTeX entry in `paper/references.bib`.

Minimum BibTeX fields to collect for each source:

- `author`
- `title`
- `journal` or `booktitle`
- `year`
- `volume` and `number`, where available
- `pages`, where available
- `doi` or `url`
